# NotebookLM向け CloudOpsクイズ作成プロンプトテンプレート

以下のプロンプトテキストをコピーして、NotebookLM（または他のLLM）に貼り付けて使用してください。

---

【目的】
アップロードされたソース資料「AWS CloudOps_BenchPrep問題集」に基づき、AWS Certified Cloud Operations Engineer - Associate (SOA-C03) 向けの練習問題を新たに作成してください。

【問題作成のガイドライン】

問題形式の再現: 
- ソース資料に見られるような、「ある企業は〜」「CloudOps エンジニアは〜」という具体的なユースケースから始まるシナリオベースの4択問題を優先して作成してください。
- 知識を問う形式や「2つ選択してください」「3つ選択してください」という複数選択形式の問題も適宜織り交ぜてください。

トーンと難易度:
- AWS 認定試験（SysOps Administrator や DevOps Engineer）のような専門的なトーンを維持してください。
- 「運用上のオーバーヘッドを最小限に抑えながら」「最もコスト効率の良い方法で」といった制約条件を必ず含めてください。
- 様々なサービスを組み合わせた超絶複雑な問題にしてください。

対象トピック: 
ソース資料に含まれる以下のトピックを網羅してください：
- AWS Systems Manager（ランブック、パッチ適用）
- CloudWatch、EventBridge（Pipes、通知）

【重要：CloudOps試験外の排除条件】
以下のタスクおよびサービスはSOA-C03試験の範囲外です。**これらを正解とする問題や、主題とする設問は必ず排除してください**：
- **除外するタスク**: 分散アーキテクチャの設計、CI/CDパイプラインの設計、ハイブリッドネットワークやマルチVPCネットワークの設計、ソフトウェア開発、セキュリティ・コンプライアンス要件の定義、ランサムウェア防御戦略の策定、リソースキャパシティーの評価・計画、コストおよびTCOの分析、請求管理。
- **除外するサービス群**: 機械学習サービス（Amazon Rekognition, Lex, Kendra, Textractなど）、高度なデータ分析（Amazon EMR, MSK, Clean Roomsなど）、フロントエンド・開発ツール（AWS AppSync, Amazon Q Developer, AWS AppConfigなど）、ビジネスアプリケーション（Amazon Connect, Pinpointなど）。
※あくまで「既存環境の運用、モニタリング、修復、自動化、ネットワークの保守、セキュリティの適用」に焦点を当ててください。

出力構成:
問題: シナリオと設問を記述。
選択肢: A, B, C, D（または必要数）。
正解: どの選択肢が正しいか。
解説: なぜその選択肢が正解なのか、他の選択肢がなぜ不適切なのかを、ソース資料の技術解説を引用しながら簡潔に説明してください。
