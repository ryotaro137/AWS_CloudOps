# 勉強日記

## 学習内容(Udemyの模試での間違い箇所)
- パブリック X.509 証明書を含む IdP メタデータとIAM Identity Center SAML メタデータのコピー
- S3 ストレージレンズダッシュボード
- S3 クロスアカウントレプリケーションのデフォルトでレプリケートされたオブジェクトの所有権は送信元バケットの所有者になる。
- 暗号化されたDBインスタンスのスナップショットも暗号化される。
- S3のアクセス許可の管理は、バケットポリシー、ACL、IAMポリシーの順で優先される。
- AWS System Mangere AutomationでAWSリソースの変更を自動化する。
- AMIは作成したリージョンにのみ保存される。異なるリージョンで使用するためにはコピーする。
- AWSマネージドキーとユーザーマネージドキーの使い分け
- AWS Global Acceleratorは、2つのグローバルIPアドレスを提供する。また、ヘルスチェックは、エンドポイントのIPアドレスではなく、ホスト名を使用する。
